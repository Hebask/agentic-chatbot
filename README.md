# Agentic Chatbot

A FastAPI-based agentic chatbot that can take actions such as creating tasks, updating task status, saving notes, and searching notes.

## Features
- FastAPI backend
- SQLite database
- SQLAlchemy ORM
- Modular architecture
- `.env` configuration
- Task and note tools
- Starter agent service

## Setup

```bash
python -m venv .venv
```

# Activate the environment, then install dependencies:
```bash
pip install -r requirements.txt
```
# Create a .env file from .env.example.

# Run the app:
``` bash
python run.py
```
# Open docs at:
http://127.0.0.1:8000/docs
